UI Exercises

Type

Design bootcamp exercises

Role

UI Design

Duration

Sep - Oct 2020 (2 months)

Tools

Figma

Description

Weekly UI exercises done as a part of Georgia Tech Bits of Good design bootcamp

Overview

For a design bootcamp I participated in, we were given a UI design prompt to work on every week. My approach to these challenges was to get outside my comfort zone and try different styles of design for each prompt.

Exercises

User Profile + Responsive Design

For my user profile practice, I wanted to create a practical and clean interface for someone to show off projects they've worked on. I worked on the mobile design earlier on in the bootcamp, and designed the web version later as part of a responsive design practice.


Login Screen

I wanted to experiment with doing dark-themed designs, and so for my login screen practice, I decided to make a red and black colored login screen that would be suitable for something like a streetwear store.


Home Screen

For my home screen design prompt, I designed a weather forecast website that provides general weather information, weekly weather information, and weather tips. I focused on creating a visually appealing website, and took a lot of inspiration from Google's weather app and weather.com.


Search Screen

I wanted to design a search screen for Google Earth that takes aspects from Google's regular search page and the current Google Earth website. A notable feature that I added was the "popular searches" and "for you" lists, inspired by MIT's website, to encourage users to explore different locations.


Settings

I wanted to experiment with the design of a children's game setting screen, which has a much different target audience and visual style compared to the designs I worked on.
